# 健身计时器

一款移动端健身训练计时工具，使用 Vue3 + Vite + TypeScript + PWA 开发。

## 功能特性

- **训练计时**：正向计时器，支持开始、暂停、重置
- **组数统计**：手动加组、减组，数据本地持久化
- **组间休息**：可自定义休息时长（默认 120 秒），倒计时结束弹出提醒 + 手机震动 + 提示音
- **模式区分**：训练模式 / 休息模式，界面样式明显区分
- **屏幕常亮**：训练时自动保持屏幕常亮，避免手机自动锁屏
- **PWA 支持**：可添加到手机桌面作为独立 App 使用，支持离线访问

## 技术栈

- Vue 3（组合式 API + `<script setup>`）
- Vite（构建工具）
- TypeScript
- PWA（Progressive Web App）
- 浏览器原生 API（Wake Lock、Vibration、Web Audio）

## 项目运行步骤

### 1. 环境准备

确保已安装 Node.js（建议版本 18+）和 npm：

```bash
node -v    # 检查 Node.js 版本
npm -v     # 检查 npm 版本
```

### 2. 安装依赖

```bash
# 进入项目目录
cd 健身计时器

# 安装依赖
npm install
```

### 3. 开发模式运行

```bash
npm run dev
```

启动后会显示本地访问地址，如：`http://localhost:5173`

**局域网访问**：项目已配置 `host: '0.0.0.0'`，同一 WiFi 下的手机可通过电脑 IP 地址访问：
- 查看电脑 IP：Windows 在命令行输入 `ipconfig`，Mac/Linux 输入 `ifconfig`
- 手机浏览器访问：`http://你的电脑IP:5173`

### 4. 生产环境构建

```bash
npm run build
```

构建产物在 `dist` 目录，可部署到任何静态文件服务器。

### 5. 预览生产版本

```bash
npm run preview
```

## PWA 添加到手机桌面教程

### Android 手机（Chrome 浏览器）

1. **打开应用**
   - 在 Chrome 浏览器中访问应用地址
   - 确保使用 HTTPS 访问（本地开发可用 HTTP）

2. **添加到主屏幕**
   - 点击浏览器右上角的「三点菜单」（⋮）
   - 选择「添加到主屏幕」或「安装应用」
   - 在弹出的对话框中点击「添加」或「安装」

3. **完成安装**
   - 应用图标会出现在手机主屏幕上
   - 点击图标即可像原生 App 一样使用

### iPhone（Safari 浏览器）

1. **打开应用**
   - 在 Safari 浏览器中访问应用地址
   - 注意：iOS 只支持 Safari 浏览器添加 PWA

2. **添加到主屏幕**
   - 点击底部的「分享按钮」（方框带向上箭头的图标）
   - 向下滑动，找到「添加到主屏幕」
   - 点击进入

3. **确认添加**
   - 可以修改显示名称
   - 点击右上角的「添加」

4. **完成安装**
   - 应用图标会出现在手机主屏幕上
   - 点击图标即可全屏使用

### 注意事项

1. **HTTPS 要求**：PWA 在生产环境需要 HTTPS 才能正常使用所有功能（如推送通知、Service Worker 等）。本地开发环境（localhost）不受此限制。

2. **浏览器兼容性**：
   - Wake Lock API（屏幕常亮）：Chrome 84+、Edge 84+
   - Vibration API（震动）：大多数 Android 浏览器支持，iOS Safari 不支持
   - Web Audio API（提示音）：所有现代浏览器支持

3. **离线使用**：首次访问后，应用会缓存资源，之后可在无网络环境下使用。

## 使用说明

### 基本操作

1. **开始训练**：点击「开始训练」按钮，计时器开始计时
2. **暂停/继续**：训练中点击「暂停」或「继续」按钮
3. **记录组数**：每完成一组，点击「+」按钮增加组数
4. **开始休息**：点击「开始休息」按钮，进入休息倒计时
5. **重置**：点击「重置」按钮，清零计时器和组数

### 设置

- 点击右上角齿轮图标打开设置
- 可自定义休息时长（30-600 秒）
- 提供常用预设：60 秒、90 秒、120 秒、180 秒

### 数据持久化

- 组数数据自动保存在浏览器本地存储中
- 休息时长设置也会自动保存
- 刷新页面或关闭浏览器后重新打开，数据不会丢失
